# 罗氏cobas8000 LIS接口开发手册

## 简介
本资源文件提供了罗氏cobas8000与LIS（实验室信息系统）之间的通讯协议的详细描述。内容涵盖了通信场景、ASTM和HL7协议格式的字段详细描述以及示例，旨在帮助开发人员理解和实现cobas8000与LIS之间的数据交换。

## 内容概述
- **通信协议概述**：介绍了cobas8000与LIS之间的通信协议的基本原理和通信场景。
- **ASTM协议**：详细描述了ASTM协议的字段定义、数据格式以及在cobas8000与LIS通信中的应用示例。
- **HL7协议**：详细描述了HL7协议的字段定义、数据格式以及在cobas8000与LIS通信中的应用示例。
- **示例代码**：提供了基于ASTM和HL7协议的通信示例代码，帮助开发人员快速上手。

## 适用对象
本手册适用于以下人员：
- 实验室信息系统（LIS）开发人员
- 罗氏cobas8000设备的技术支持人员

## 使用说明
1. **阅读协议概述**：首先阅读通信协议概述部分，了解cobas8000与LIS之间的通信原理和场景。
